Platform Overview

A ReAct orchestrator discovers your data landscape and builds domain knowledge from documentation, then delegates to specialized agents—an Analysis Agent for data fetching and transformation, a CodeAct Agent for autonomous code generation and execution in isolated sandboxes. Full observability across every reasoning step and action.

Documentation-First

Reads Product Requirement Prompts (PRPs) from MCP servers to understand data domains, entity relationships, and API endpoint structures before fetching data.

MCP Integration

Connect any data source through Model Context Protocol servers. Each MCP server provides documentation resources that teach the agent about your data domain.

Smart Data Fetching

Constructs complete API URLs based on documentation understanding. Automatically converts JSON responses to CSV and caches data for efficient reuse.

Secure Sandboxes

Executes Python analysis in isolated E2B sandboxes with pandas, numpy, matplotlib, and seaborn pre-installed. CSV files automatically mounted for instant access.

Natural Language

Ask questions in plain English. The agent orchestrates multiple tools, reads documentation, fetches data, writes Python code, and presents insights—all conversationally.

Auto Visualization (Coming Soon)

Real-time chart generation and inline display within the conversational interface. Matplotlib and seaborn visualizations will be automatically embedded with full analysis context.

The Agent Workflow

A seamless flow from your question to actionable insights. The agent orchestrates data discovery, fetching, analysis, and response generation autonomously.

Step 1

User Query

You ask a question in natural language. The agent receives your query and conversation history.

Step 2

Resource Discovery

Agent discovers available data sources via MCP servers and reads documentation to understand the domain.

Step 3

Data Fetching

Agent constructs API queries based on documentation and fetches data from external sources.

Step 4

Analysis Execution

Agent executes Python code in secure sandbox to analyze data, generate statistics, and create visualizations.

Step 5

Response Generation

Agent synthesizes insights and streams a formatted response with markdown, insights, and visualizations.
